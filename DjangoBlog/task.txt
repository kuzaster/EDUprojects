1) Авторизация
Сделать регистрацию пользователей, вход на сайт в шапке сайта (если пользователь уже залогинен, то там будет кнопка «Выход»). Поля для регистрации/входа: email, пароль. Пользовать после авторизации попадает в личный кабинет.

2) Личный кабинет
Страницы личного кабинета: личный профиль, публикации.

Личный профиль
Поля:
Имя
Фамилия
Аватар
Email (неизменяемое поле)
Телефон
Skype

Публикации
Список публикаций пользователя. Публикацию можно удалить, редактировать.
Поля:
Название публикации (является ссылкой на детализацию публикации)
Дата публикации
Короткое описание (показывать только на странице списка публикаций)
Описание (показывать только на странице детализации публикации)

3) Публикации пользователей
Список публикаций пользователей по типу публикации в личном кабинете с дополнительным полем «Автор». Публикации можно сортировать по дате размещения, по автору. Публикации можно комментировать (к своей публикации оставлять комментерии нельзя).
